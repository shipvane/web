# Shipvane

**Ship your code, just point it the right direction.**

Shipvane turns a ticket into a merge-ready PR — and nothing ships off-course.
Every change is driven by a **real CI gate** (it's not done until the build is
green) and **your approval** (you point the direction; Shipvane does the work).

> 🚧 **Early development.** This repository reserves the `shipvane` name across
> npm, PyPI, and GitHub while the product is built. Docs and the CLI are coming
> soon — https://shipvane.com

## Where to find it
- **Web:** https://shipvane.com
- **npm:** `npm i shipvane` *(reserved)*
- **PyPI:** `pip install shipvane` *(reserved)*
